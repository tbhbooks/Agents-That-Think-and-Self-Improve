================================================================================
COMMON ERRORS
================================================================================

ERROR: AuthenticationError
  → API key not set. Run: export ANTHROPIC_API_KEY="sk-ant-..."

ERROR: ModuleNotFoundError: No module named 'anthropic'
  → SDK not installed. Run: pip install anthropic
  → Or virtual environment not activated. Run: source .venv/bin/activate

ERROR: ConnectionError
  → No internet. Or if using Ollama, it's not running: ollama serve
